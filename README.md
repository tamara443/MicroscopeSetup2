# MicroscopeSetup

# Technical
- Python 3.10.8
- Packages installed through requirements.txt
- .dll for the Laser and MCLNanoDrive Stage are provided in this repository and referenced relatively in code. Please keep in mind, that for guaranteed usage the dll's should be installed on the system, provided with the manufactures installer. Although this should be fine here, because there is no changing hardware and the setup static.
- Install USB Driver for Yokogawa GS200 SMU: http://sine.ni.com/apps/utf8/niid_web_display.download_page?p_id_guid=7B39644C1D1D4C79E04400144F1EF859 -> Easily through NI MAX Package Manager https://knowledge.ni.com/KnowledgeArticleDetails?id=kA03q000000YGQwCAO&l=de-DE
